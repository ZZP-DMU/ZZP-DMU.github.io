Latex中使用gnuplot绘图

Introduce

  1. 添加xelatex编译参数-shell-escape
  2. 使用gnuplottex宏包，\usepackage[miktex]{gnuplottex}
  3. 使用过环境
\begin{gnuplot}[terminal=epslatex]
*----gnuplot command----*
\end{gnuplot}

Example:

\documentclass{ctexart}
\usepackage[miktex]{gnuplottex}
\begin{document}
\begin{gnuplot}[terminal=epslatex]
set pm3d  
set isosamples 50,50 
splot x**3+y**2 
\end{gnuplot}
\end{document}
