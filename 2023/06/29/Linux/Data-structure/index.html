算法复杂度

时间复杂度

大O表示法

T(n)=O(f(n)),f(n)为每行代码的执行次数，O表示正比例关系。这个公式称为：算法的渐进时间复杂度。这个公式并不是用来表示算法真实的运行时间，而是表示算法随变量n变化的趋势。

  1. O(1)常数阶：没有循环等复杂结构，代码的复杂度都是O(1)。
  2. O(n)线性阶：常见在for循环，其中的代码会执行n遍，因此其消耗的时间随n变化而变化。
  3. O(nk)k次方阶：循环嵌套k次就会得到。
  4. O(logN)对数阶：循环次数,那么
  5. O(nlogN)线性对数阶:循环内嵌套对数阶就可得到线性对数阶复杂度。

组合式复杂度分析

  1. 取最大复杂度作为整个算法复杂度
  2. 取多个复杂度之和作为整个算法的复杂度

时间复杂度类型

  1. 最好时间复杂度：代码执行次数最少的情况
  2. 最坏时间复杂度：遍历算法中所有或者最多的状态
  3. 平均时间复杂度：看不懂
  4. 均摊时间复杂度：看不懂

空间复杂度

空间复杂度全称就是渐进空间复杂度，用来表示算法的存储空间与数据规模之间的增长关系。和时间复杂度一样，空间复杂度也是用大 O 进行表示。

  1. 如果申请的是有限个数（常量）的变量，空间复杂度为 O(1)。
  2. 如果申请的是一维数组，队列或者链表等，那么空间复杂度为 O(n)。
  3. 如果申请的是二维数组，那么空间复杂度为 O(n²)。

链表

定义

单向链表，又名列表，是一种递归的数据结构。。一个单向链表或为空，或包含一个元素(key)和一个子链表(next)。

同时我们也可以认为一个链表有若干节点组成，每一个节点包括一个值和一个指向子链表的引用。如图

我们不难发现，节点在某种意义上就是一个链表，换一种说法就是节点是链表的外在具体表现。当我们确定一个节点时，我们就得到了一个元素和子链表，也就确定了一个链表。

    struct list *next;

获取长度

根据定义，空链表长度为0，非空链表的长度为除去第一个元素的子链表长度加一，即$$ length(x:xs)=length(xs)+1 $$
因为遍历计算链表长度，所以其时间复杂度是O(1)。

int length(struct list X)
    int n=0;
    while X.next != NULL
    { 
        n=n+1;
        X=tail(X);
    }
    return n+1;

反向索引

一般形式是获取链表中倒数第i个元素，last是反向索引的一种特例。

  1. 方法一:遍历两边，第一次获取链表长度n，第二次获取第n-i-1个元素，即$$lastat(i,X)= getat((length(X)-i-1),X)$$
  2. 方法二：滑动窗口法，即使用相距i步的指针，首先去除链表X的前i个节点，然后将链表X和截取后的链表同时取tail直至截取后的链表为空，那么就将获得反向索引的节点。如下$$lastat(i,X)=slide(X,drop(i,X))$$
    其中$$slide(x:xs,[y])=x$$
    $$slide(x:xs,y:ys)=slide(xs,ys)$$
    函数drop(m,X)丢弃链表X前m个元素，如下$$ drop(0,xs)=xs$$
    $$ drop(m,NIL)=NIL $$
    $$ drop(m,x:xs)=drop(x-1,xs)$$
function last-at(i,X)
    p=X
    while i>0 do
        X = tail(X)
        i = i - 1
    while tail(X) != NIL do
        X = tail(X)
        p = tail(p)
    return head(p)

添加

添加称作append，和cons操作对称，cons在表头增加，append在末尾增加。由于要遍历链表，所以其时间复杂度为O(n)。有以下规律$$ append(NIL,x)=x:NIL$$ $$append(y:ys,x)=y:(append(ys,x))$$

struct list *append(struct list *X,int x)
{
    struct list _X;
    _X.key = x;
    _X.next = NULL;
    if X->next == NULL
    {
        X->next = &(_X);
        return X;
    }   
    struct list *H;
    H = X;
    while(X->next != NULL)
    {
        X = tail(*X);
    }
    X->next=&(_X);
    X = H;
    return X;
}
